About

My path into development has been intentional. My background in Information System Security taught me to think in systems and structure, while psychology shaped how I understand people — how they interact and experience technology.

I'm confident in HTML and CSS, actively deepening my understanding of JavaScript and React. I focus on strong fundamentals, debugging, problem-solving, and writing clean, maintainable UI code that prioritizes user experience.
